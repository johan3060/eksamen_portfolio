Tema 1

Grundlæggende Web

I tema 1 lærte vi grundlæggende om html og css, flexbox, stilarter og hvordan man gør sin side responsiv i forhold til forskellige størrelser af pc'er og telefoner vha. media queries. Projektet gennem hele temaet var vores stilartssite, som vi gradvist byggede videre på.

Tema 2

Animation

I dette tema lærte vi bl.a. om skitsering og ide-genering, som jeg benyttede mig af til at lave opgaven. Vi benyttede os af illustrator til at lave vores assets, og vi lærte hvordan man rentegnede skitser i illustrator.

● 02.01.04 - Assetliste og designdokumentation

Til animations-opgaven brugte vi meget tid på at lære hvordan javascript fungerede, samt hvordan vi lavede css-animationer. Vi lærte om hvordan man redigerede og implementerede lyd, og hvordan man kunne programmere spillet til at overholde reglerne (f.eks. stop efter x antal tid).

● 03.03.04 - Animations-opgaven

I gruppeprojektet benyttede vi os af meget af det vi lærte til vores eget spil. Da projektet vi lavede var i samarbejde med privatsnak.dk benyttede vi os den samme farvepallete til forsiden. Vi beskæftigede vi os med aktivitets-, layout- og statemachine-diagrammer. Desuden benyttede vi os af en test på en teenager til at hjælpe os med at finde vores stilart.

● 02.05.03 - Færdig opgave

Tema 3

Grundlæggende video

Til pilotprojektet lærte vi at bruge premiere pro. Vi lærte hvordan vi klippede i film, og hvordan vi tilføjede og redigerede i lyden. Vi lærte desuden om klippeteknik og billedkomposition - hvordan billedet skulle frames. Vi benyttede os af ekstern lyd, da det er nemmest at bruge i redigeringsprocessen.

● 03.01.01 - Pilotprojekt

Til virksomhedsvideon benyttede vi os af mange af de samme kompetencer som vi lærte til pilotprojektet. Vi lavede et storyboard til at til at hjælpe os med de shots vi gerne ville have, og havde fokus på det gyldne snit i interviewet.

● 03.02.05 - Virksomhedsvideo

Til redesign-sitet benyttede vi os af html og css. Vi lærte om burger-menuer, som vi benyttede os af til mobil-versionen af vores site. Desuden lærte vi om github.
